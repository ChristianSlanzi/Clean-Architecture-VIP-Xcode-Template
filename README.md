# Clean-Architecture-VIP-Template
Clean Architecture VIP Template

Swift 5
Xcode 10.2+

How to use:

clone this repo in `cd ~/Library/Developer/Xcode/Templates`
This template is required `Alamofire and SwiftyJSON`
